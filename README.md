# Simple Banking System (C++)

This project is a simple console-based banking system developed in C++.
It allows users to create an account, deposit money, withdraw money, and check their balance.

## Features
- Create bank account
- Deposit funds
- Withdraw funds
- Display account balance

## Technologies
- C++
- GitHub for version control

## How to Run
1. Clone the repository
2. Compile using a C++ compiler (g++)
3. Run the executable

## Collaboration
This project was developed collaboratively using GitHub branches and pull requests.
